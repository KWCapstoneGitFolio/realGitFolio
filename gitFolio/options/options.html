데이터 자동 업데이트 사용

GitHub 페이지 방문 시 자동으로 데이터를 업데이트합니다. 비활성화하면 수동으로 새로고침해야 합니다.

저장소 분석 시 기본으로 입력될 GitHub 사용자명을 입력하세요.

비공개 저장소 접근 및 API 요청 제한 증가를 위해 GitHub 토큰이 필요합니다. 여기에서 토큰 생성

필요한 권한: repo, read:user

토큰은 안전하게 저장되지만, 공유 기기를 사용하는 경우 '이 기기에만 저장'을 권장합니다.

GitFolio UI를 표시할 방식을 선택하세요. 사이드바 모드는 화면 오른쪽에 고정됩니다.
